# ReadMe - Power BI Enedis

## I. Présentation du projet
Il porte sur l’analyse et la visualisation des **DPE (Diagnostics de Performance Énergétique)** à l’aide de **Power BI** à partir d’un fichier CSV partagé.
Le travail a été effectué en présentiel à l’IUT, avec un partage des données via **OneDrive**, ce qui explique l’usage limité des branches Git.
